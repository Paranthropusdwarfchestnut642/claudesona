# 🎭 claudesona - Add expressive character sprites to Claude

[![](https://img.shields.io/badge/Download-claudesona-blue)](https://github.com/Paranthropusdwarfchestnut642/claudesona)

## 📖 About this project

Claudesona Emotion Sprites acts as a browser extension for your claude.ai sessions. The tool scans text outputs for specific tags. It detects emotion markers like happy, thoughtful, or skeptical within the conversation flow. Once the extension finds a tag, it displays a small image. These images are sprite files. Each sprite measures 128x128 pixels. The images have transparent backgrounds. This creates a visual layer for your AI interactions.

## 🛠 Features

*   **Real-time detection:** The software monitors text input for tags in the background.
*   **Automatic replacement:** The program swaps text codes for visual assets without manual input.
*   **Small footprint:** The files use minimal resources. 
*   **Visual feedback:** Expressions change based on the mood tags included in your prompts.
*   **High transparency:** Images blend into your browser theme.

## 📋 System requirements

*   **Operating System:** Windows 10 or Windows 11.
*   **Browser:** Google Chrome, Microsoft Edge, or Brave.
*   **Memory:** 4GB RAM minimum.
*   **Storage:** 50MB free disk space.
*   **Internet:** Stable connection for browser synchronization.

## 🚀 Getting Started

Follow these steps to set up the extension on your computer.

1. Visit this page to download: [https://github.com/Paranthropusdwarfchestnut642/claudesona](https://github.com/Paranthropusdwarfchestnut642/claudesona)
2. Locate the green Code button on the repository page.
3. Select Download ZIP from the dropdown menu.
4. Open your Downloads folder.
5. Right-click the claudesona file folder.
6. Select Extract All to create a readable folder on your hard drive.
7. Open your web browser.
8. Type `chrome://extensions` in the address bar and press Enter.
9. Toggle the Developer mode switch in the top right corner.
10. Click Load unpacked.
11. Choose the folder you extracted earlier.

## ⚙️ How to use the extension

Once you install the software, it starts working. Open claude.ai in your browser. Type your usual prompts. Include the emotion tags within your text to trigger the sprites. 

Available tags include:
*   `<claude_happy />`
*   `<claude_thoughtful />`
*   `<claude_skeptical />`

The extension scans the response for these markers. It pulls the matching sprite image. The image appears in place of the text command. You do not need to refresh the page for the sprites to update.

## 🔧 Frequently asked questions

**Does this change the AI responses?**
No. The tool only changes what you see on your screen. The base AI model remains unchanged.

**Will this slow down my browser?**
The extension uses minimal memory. Most modern computers will not notice a change in speed.

**Can I create my own sprites?**
The software supports local image files. You can replace the default images in the assets folder. Keep your files at 128x128 pixels for the best display quality.

**Do I need an account?**
You only need an account for claude.ai. The extension function works locally in your browser.

**Can I remove the extension?**
Yes. Return to the extensions page in your browser. Click Remove on the entry for claudesona. Delete the folder you saved to your computer.

## 🛡 Security and Privacy

This program processes information locally. It does not send your chat data to external servers. The code monitors your browser tabs to identify the specific tags mentioned. It does not read your private account details or banking data. All visual assets install directly to your device. You control the software through your browser settings. 

## 🩺 Troubleshooting

If the sprites do not appear, check these items:

1.  **Check Developer Mode:** Ensure the toggle is still active in your extension settings.
2.  **Verify Tags:** Look for typos in your emotion tags. They must match the format exactly including brackets and slashes.
3.  **Refresh:** If the extension fails to load, refresh your chat tab.
4.  **Reinstall:** Remove the extension from your browser. Load the extracted folder again.
5.  **Browser Update:** Keep your browser current to ensure compatibility with extension APIs.
